# CNode社区网页客户端 —— React版

### 使用方法

* 需手动去往 `app/components/TopicDetail.js` 、 `app/package.json -- script -- start` 以及 `app/components/TopicList.js` 里将hostname设置为你本地的IP地址

* 打开客户端

``` sh
$ cd app
$ npm i
$ npm run build
$ npm run start
```

* 打开爬虫服务器

``` sh
$ cd server
$ npm i
$ node app.js
```

### 遗留问题

* 需手动设置IP
